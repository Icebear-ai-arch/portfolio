# 3D Portfolio (Justfer F. Carabuena)

This repo contains a Vite + React portfolio site. Below are steps to build and deploy it.

Local build
```
npm ci
npm run build
npm run preview
```

Deploy options (free)

- **Vercel (recommended)**
  - Create a GitHub repo and push this project.
  - Import the repo on https://vercel.com and choose the Vite preset.

- **GitHub Pages (via Actions)**
  - This repository includes a GitHub Actions workflow that builds and deploys `dist/` to GitHub Pages when you push to `main`.
  - Create a GitHub repo, push to `main`, then enable GitHub Pages for the repository (if needed).

- **Netlify**
  - Connect the repo and set `npm run build` as the build command and `dist` as the publish directory.

Notes
- If you deploy to a repository pages URL (username.github.io/repo-name), set `base` in `vite.config.js` to `/repo-name/`.
- Static assets should live in `public/`.
